# 02 - Binario para decimal

**Nivel:** Pleno  
**Categoria:** Operadores

## Enunciado

Crie um programa que receba um numero escrito em binario e mostre o seu valor em decimal, usando apenas operacoes matematicas.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `1010` | `10` |
| 2 | `1111` | `15` |
| 3 | `100000` | `32` |
