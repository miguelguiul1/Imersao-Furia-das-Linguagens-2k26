# 08 - Busca binaria

**Nivel:** Pleno  
**Categoria:** Vetor

## Enunciado

Crie um programa que receba uma lista ja ordenada e um valor X, e mostre a posicao de X ou -1, usando busca binaria.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `[1,3,5,7,9], X=7` | `3` |
| 2 | `[2,4,6], X=5` | `-1` |
| 3 | `[10], X=10` | `0` |
