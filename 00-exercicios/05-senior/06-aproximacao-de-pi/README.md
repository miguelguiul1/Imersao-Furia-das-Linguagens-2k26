# 06 - Aproximacao de Pi

**Nivel:** Senior  
**Categoria:** Repeticao

## Enunciado

Crie um programa que receba um numero de termos e aproxime o valor de Pi pela serie de Leibniz, mostrando 4 casas decimais.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `1 termo` | `4.0000` |
| 2 | `10 termos` | `3.0418` |
| 3 | `100000 termos` | `3.1416` |
