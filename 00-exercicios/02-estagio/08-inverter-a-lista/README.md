# 08 - Inverter a lista

**Nivel:** Estagio  
**Categoria:** Vetor

## Enunciado

Crie um programa que receba uma lista e mostre-a na ordem inversa.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `[1, 2, 3]` | `[3, 2, 1]` |
| 2 | `[4, 5]` | `[5, 4]` |
| 3 | `[9]` | `[9]` |

## Dica

Percorra a lista do ultimo elemento ate o primeiro.
