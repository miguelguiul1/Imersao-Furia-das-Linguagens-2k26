# 10 - Funcao maximo divisor comum

**Nivel:** Junior  
**Categoria:** Funcao

## Enunciado

Crie uma funcao que devolva o maior divisor comum entre dois numeros, e mostre o resultado.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `12, 8` | `4` |
| 2 | `100, 10` | `10` |
| 3 | `7, 3` | `1` |

## Assinatura sugerida

`int mdc(int a, int b)`

## Dica

Use o algoritmo de Euclides.
