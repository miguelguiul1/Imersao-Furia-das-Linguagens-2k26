# 09 - Funcao dobro

**Nivel:** Iniciante  
**Categoria:** Funcao

## Enunciado

Crie uma funcao que devolva o dobro de um numero. Use-a para mostrar o dobro de um valor recebido.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `4` | `8` |
| 2 | `0` | `0` |
| 3 | `-3` | `-6` |

## Assinatura sugerida

`int dobro(int n)`

## Dica

A funcao deve receber n e retornar n * 2.
