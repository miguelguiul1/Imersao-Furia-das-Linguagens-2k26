# 01 - Soma de dois numeros

**Nivel:** Iniciante  
**Categoria:** Operadores

## Enunciado

Crie um programa que receba dois numeros inteiros e mostre a soma dos dois.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `5, 3` | `8` |
| 2 | `10, 20` | `30` |
| 3 | `-4, 4` | `0` |

## Dica

Use o operador de adicao (+) entre as duas variaveis.
