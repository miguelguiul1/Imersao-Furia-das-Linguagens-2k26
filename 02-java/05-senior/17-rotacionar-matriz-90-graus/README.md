# 17 - Rotacionar matriz 90 graus

**Nivel:** Senior  
**Categoria:** Matriz

## Enunciado

Crie um programa que receba uma matriz quadrada e mostre-a rotacionada 90 graus no sentido horario.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `[[1,2],[3,4]]` | `[[3,1],[4,2]]` |
| 2 | `[[1,2,3],[4,5,6],[7,8,9]]` | `[[7,4,1],[8,5,2],[9,6,3]]` |
| 3 | `[[5]]` | `[[5]]` |
