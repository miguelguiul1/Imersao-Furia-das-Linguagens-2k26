# 06 - Soma dos digitos

**Nivel:** Junior  
**Categoria:** Repeticao

## Enunciado

Crie um programa que receba um numero e mostre a soma de todos os seus digitos.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `123` | `6` |
| 2 | `999` | `27` |
| 3 | `5` | `5` |

## Dica

Use resto por 10 para pegar o ultimo digito e divisao inteira por 10 para remove-lo.
