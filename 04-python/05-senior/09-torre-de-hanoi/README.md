# 09 - Torre de Hanoi

**Nivel:** Senior  
**Categoria:** Funcao

## Enunciado

Crie um programa que resolva a Torre de Hanoi para N discos e informe o numero minimo de movimentos. Divida em funcoes.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `1` | `1 movimento` |
| 2 | `3` | `7 movimentos` |
| 3 | `4` | `15 movimentos` |

## Assinatura sugerida

`int pedirQuantidadeDiscos(); long hanoi(int n); void apresentarMovimentos(long total)`
