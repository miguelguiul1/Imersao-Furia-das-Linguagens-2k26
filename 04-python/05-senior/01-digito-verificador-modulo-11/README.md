# 01 - Digito verificador modulo 11

**Nivel:** Senior  
**Categoria:** Operadores

## Enunciado

Crie um programa que receba uma sequencia de 8 digitos e calcule o digito verificador pelo modulo 11.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `12345678` | `9` |
| 2 | `11111111` | `0` |
| 3 | `87654321` | `2` |
