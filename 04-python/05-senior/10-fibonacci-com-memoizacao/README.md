# 10 - Fibonacci com memoizacao

**Nivel:** Senior  
**Categoria:** Funcao

## Enunciado

Crie um programa que devolva o n-esimo numero de Fibonacci de forma eficiente, guardando resultados ja calculados.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `10` | `55` |
| 2 | `0` | `0` |
| 3 | `50` | `12586269025` |

## Assinatura sugerida

`int pedirPosicao(); long fib(int n); void apresentarFibonacci(long valor)`
