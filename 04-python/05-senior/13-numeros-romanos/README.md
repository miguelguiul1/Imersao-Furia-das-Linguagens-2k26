# 13 - Numeros romanos

**Nivel:** Senior  
**Categoria:** Numero

## Enunciado

Crie um programa que receba um numero inteiro de 1 a 3999 e o converta para algarismos romanos.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `9` | `IX` |
| 2 | `2024` | `MMXXIV` |
| 3 | `3999` | `MMMCMXCIX` |
