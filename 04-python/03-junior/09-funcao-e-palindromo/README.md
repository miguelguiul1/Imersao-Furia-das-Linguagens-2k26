# 09 - Funcao e palindromo

**Nivel:** Junior  
**Categoria:** Funcao

## Enunciado

Crie uma funcao que devolva verdadeiro se um texto for igual lido de tras para frente, e mostre o resultado.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `"arara"` | `verdadeiro` |
| 2 | `"ovo"` | `verdadeiro` |
| 3 | `"casa"` | `falso` |

## Assinatura sugerida

`bool ehPalindromo(string texto)`

## Dica

Compare o primeiro caractere com o ultimo, o segundo com o penultimo, e assim por diante.
