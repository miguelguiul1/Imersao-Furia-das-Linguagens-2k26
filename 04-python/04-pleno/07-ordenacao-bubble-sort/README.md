# 07 - Ordenacao bubble sort

**Nivel:** Pleno  
**Categoria:** Vetor

## Enunciado

Crie um programa que receba uma lista de numeros e mostre-a ordenada em ordem crescente, implementando a ordenacao sem usar funcoes prontas.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `[5, 2, 8, 1]` | `[1, 2, 5, 8]` |
| 2 | `[3, 3, 1]` | `[1, 3, 3]` |
| 3 | `[9]` | `[9]` |
