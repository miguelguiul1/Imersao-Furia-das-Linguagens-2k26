# 10 - Contar primos ate N

**Nivel:** Pleno  
**Categoria:** Funcao

## Enunciado

Crie um programa que conte quantos numeros primos existem de 2 ate um limite. Separe em funcoes de entrada, processamento e saida.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `10` | `4` |
| 2 | `2` | `1` |
| 3 | `20` | `8` |

## Assinatura sugerida

`int pedirLimite(); bool ehPrimo(int n); int contarPrimosAte(int limite); void apresentarTotal(int total)`
