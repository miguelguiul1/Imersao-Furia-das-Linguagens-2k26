# 08 - Contar ocorrencias

**Nivel:** Iniciante  
**Categoria:** Vetor

## Enunciado

Crie um programa que receba uma lista de numeros e um valor X, e conte quantas vezes X aparece na lista.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `[1,2,2,3,2], X=2` | `3` |
| 2 | `[5,5,5], X=5` | `3` |
| 3 | `[1,2,3], X=9` | `0` |

## Dica

Percorra a lista e some 1 num contador toda vez que o item for igual a X.
