# 04 - O maior de dois

**Nivel:** Iniciante  
**Categoria:** Condicao

## Enunciado

Crie um programa que receba dois numeros e mostre o maior deles. Se forem iguais, mostre IGUAIS.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `3, 9` | `9` |
| 2 | `10, 2` | `10` |
| 3 | `5, 5` | `IGUAIS` |

## Dica

Compare os dois com o operador maior que (>) e trate o caso de igualdade.
