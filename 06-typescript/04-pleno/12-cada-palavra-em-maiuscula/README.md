# 12 - Cada palavra em maiuscula

**Nivel:** Pleno  
**Categoria:** Texto

## Enunciado

Crie um programa que receba uma frase e devolva-a com a primeira letra de cada palavra em maiuscula.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `"ola mundo bom"` | `"Ola Mundo Bom"` |
| 2 | `"teste final"` | `"Teste Final"` |
| 3 | `"a b c"` | `"A B C"` |
