# 04 - Movimento do cavalo

**Nivel:** Senior  
**Categoria:** Condicao

## Enunciado

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `e4 -> f6` | `VALIDO` |
| 2 | `b1 -> c3` | `VALIDO` |
| 3 | `e4 -> e5` | `INVALIDO` |
