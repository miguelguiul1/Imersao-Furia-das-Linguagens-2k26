# 07 - Media da lista

**Nivel:** Estagio  
**Categoria:** Vetor

## Enunciado

Crie um programa que receba uma lista de numeros e mostre a media dos valores.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `[10, 8, 6]` | `8` |
| 2 | `[5, 5]` | `5` |
| 3 | `[7]` | `7` |

## Dica

Some todos os elementos e divida pela quantidade de itens.
