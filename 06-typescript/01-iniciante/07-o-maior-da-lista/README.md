# 07 - O maior da lista

**Nivel:** Iniciante  
**Categoria:** Vetor

## Enunciado

Crie um programa que receba uma lista de numeros e mostre o maior valor dela.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `[3, 7, 2, 9, 4]` | `9` |
| 2 | `[10]` | `10` |
| 3 | `[-1, -5, -3]` | `-1` |

## Dica

Guarde o primeiro item como maior e troque quando encontrar algo maior.
