# 05 - Contagem de 1 ate N

**Nivel:** Iniciante  
**Categoria:** Repeticao

## Enunciado

Crie um programa que receba um numero N e mostre todos os numeros de 1 ate N.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `5` | `1 2 3 4 5` |
| 2 | `3` | `1 2 3` |
| 3 | `1` | `1` |

## Dica

Use um laco que comeca em 1 e vai somando 1 ate chegar em N.
