# 01 - Calculo do IMC

**Nivel:** Junior  
**Categoria:** Operadores

## Enunciado

Crie um programa que receba o peso (kg) e a altura (m) de uma pessoa e mostre o IMC com 2 casas decimais.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `70, 1.75` | `22.86` |
| 2 | `90, 1.80` | `27.78` |
| 3 | `50, 1.60` | `19.53` |

## Dica

IMC = peso / (altura * altura). Arredonde para 2 casas.
