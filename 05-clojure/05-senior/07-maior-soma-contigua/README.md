# 07 - Maior soma contigua

**Nivel:** Senior  
**Categoria:** Vetor

## Enunciado

Crie um programa que receba uma lista de numeros e mostre a maior soma possivel de uma sequencia de elementos vizinhos.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `[-2,1,-3,4,-1,2,1,-5,4]` | `6` |
| 2 | `[1, 2, 3]` | `6` |
| 3 | `[-5, -2, -8]` | `-2` |
