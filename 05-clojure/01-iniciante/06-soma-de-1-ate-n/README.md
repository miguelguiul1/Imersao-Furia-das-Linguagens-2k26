# 06 - Soma de 1 ate N

**Nivel:** Iniciante  
**Categoria:** Repeticao

## Enunciado

Crie um programa que receba N e mostre a soma de todos os numeros de 1 ate N.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `5` | `15` |
| 2 | `3` | `6` |
| 3 | `10` | `55` |

## Dica

Crie uma variavel de soma iniciada em 0 e acumule cada numero dentro do laco.
