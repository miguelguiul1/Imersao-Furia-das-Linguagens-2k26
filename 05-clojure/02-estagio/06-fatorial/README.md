# 06 - Fatorial

**Nivel:** Estagio  
**Categoria:** Repeticao

## Enunciado

Crie um programa que receba N e mostre o fatorial de N.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `5` | `120` |
| 2 | `3` | `6` |
| 3 | `0` | `1` |

## Dica

Multiplique todos os numeros de 1 ate N. O fatorial de 0 e 1.
