# 03 - Categoria do nadador

**Nivel:** Junior  
**Categoria:** Condicao

## Enunciado

Crie um programa que receba a idade de um atleta e o classifique: INFANTIL, JUVENIL, ADULTO ou SENIOR.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `8` | `INFANTIL` |
| 2 | `14` | `JUVENIL` |
| 3 | `65` | `SENIOR` |

## Dica

Trate as faixas em ordem crescente de idade.
