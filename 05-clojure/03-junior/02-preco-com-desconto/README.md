# 02 - Preco com desconto

**Nivel:** Junior  
**Categoria:** Operadores

## Enunciado

Crie um programa que receba o preco de um produto e um percentual de desconto, e mostre o preco final.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `100, 10%` | `90` |
| 2 | `250, 50%` | `125` |
| 3 | `80, 0%` | `80` |

## Dica

Preco final = preco - preco * (percentual / 100).
