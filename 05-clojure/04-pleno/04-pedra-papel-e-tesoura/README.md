# 04 - Pedra papel e tesoura

**Nivel:** Pleno  
**Categoria:** Condicao

## Enunciado

Crie um programa que receba as jogadas de dois jogadores e diga o resultado: JOGADOR 1, JOGADOR 2 ou EMPATE.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `PEDRA vs TESOURA` | `JOGADOR 1` |
| 2 | `PAPEL vs TESOURA` | `JOGADOR 2` |
| 3 | `PEDRA vs PEDRA` | `EMPATE` |
