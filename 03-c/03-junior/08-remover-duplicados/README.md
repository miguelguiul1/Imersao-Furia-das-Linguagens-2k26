# 08 - Remover duplicados

**Nivel:** Junior  
**Categoria:** Vetor

## Enunciado

Crie um programa que receba uma lista e mostre-a sem valores repetidos, mantendo a ordem de aparicao.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `[1, 2, 2, 3, 1]` | `[1, 2, 3]` |
| 2 | `[5, 5, 5]` | `[5]` |
| 3 | `[1, 2, 3]` | `[1, 2, 3]` |

## Dica

Monte uma nova lista adicionando cada item so se ele ainda nao estiver nela.
