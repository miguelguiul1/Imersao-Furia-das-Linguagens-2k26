# 07 - O segundo maior

**Nivel:** Junior  
**Categoria:** Vetor

## Enunciado

Crie um programa que receba uma lista de numeros e mostre o segundo maior valor dela.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `[3, 7, 2, 9, 4]` | `7` |
| 2 | `[10, 10, 5]` | `10` |
| 3 | `[1, 2]` | `1` |

## Dica

Encontre o maior; depois procure o maior entre os que sao diferentes dele.
