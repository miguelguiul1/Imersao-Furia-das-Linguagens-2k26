# 04 - Tipo de triangulo

**Nivel:** Junior  
**Categoria:** Condicao

## Enunciado

Crie um programa que receba os 3 lados de um triangulo e diga se ele e EQUILATERO, ISOSCELES, ESCALENO ou NAO FORMA.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `3, 3, 3` | `EQUILATERO` |
| 2 | `5, 5, 8` | `ISOSCELES` |
| 3 | `1, 1, 10` | `NAO FORMA` |

## Dica

Primeiro verifique se cada lado e menor que a soma dos outros dois.
