# 09 - Potencia recursiva

**Nivel:** Pleno  
**Categoria:** Funcao

## Enunciado

Crie um programa que calcule a base elevada ao expoente sem usar operador de potencia. Organize a solucao em funcoes.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `2, 10` | `1024` |
| 2 | `5, 0` | `1` |
| 3 | `3, 3` | `27` |

## Assinatura sugerida

`int pedirExpoente(); long potencia(int base, int expoente); void apresentarResultado(long resultado)`
