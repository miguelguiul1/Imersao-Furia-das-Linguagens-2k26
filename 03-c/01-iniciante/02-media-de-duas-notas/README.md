# 02 - Media de duas notas

**Nivel:** Iniciante  
**Categoria:** Operadores

## Enunciado

Crie um programa que receba duas notas e mostre a media entre elas.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `8, 6` | `7` |
| 2 | `10, 5` | `7.5` |
| 3 | `7, 7` | `7` |

## Dica

Some as duas notas e divida o resultado por 2.
