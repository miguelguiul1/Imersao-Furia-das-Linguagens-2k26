# 09 - Funcao e primo

**Nivel:** Estagio  
**Categoria:** Funcao

## Enunciado

Crie uma funcao que devolva verdadeiro se um numero for primo, e mostre o resultado.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `7` | `verdadeiro` |
| 2 | `10` | `falso` |
| 3 | `1` | `falso` |

## Assinatura sugerida

`bool ehPrimo(int n)`

## Dica

Teste divisores de 2 ate n-1.
