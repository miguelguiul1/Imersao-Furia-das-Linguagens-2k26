# 02 - Contar bits ligados

**Nivel:** Senior  
**Categoria:** Operadores

## Enunciado

Crie um programa que receba um numero inteiro e conte quantos bits iguais a 1 ele tem na representacao binaria, usando operacoes bit a bit.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `7` | `3` |
| 2 | `8` | `1` |
| 3 | `255` | `8` |
