# 15 - Idade exata

**Nivel:** Senior  
**Categoria:** Data

## Enunciado

Crie um programa que receba a data de nascimento e a data de hoje, e mostre a idade exata em anos, meses e dias.

| Teste | Entrada | Saida esperada |

| 3 | `10/12/2020 ate 05/03/2024` | `3 anos, 2 meses, 24 dias` |
