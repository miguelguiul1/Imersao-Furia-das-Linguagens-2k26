# 12 - Compactacao por contagem

**Nivel:** Senior  
**Categoria:** Texto

## Enunciado

Crie um programa que receba um texto e devolva sua versao compactada, trocando sequencias repetidas pela letra seguida da quantidade.

## Cenarios de teste

| Teste | Entrada | Saida esperada |

| 1 | `"aaabb"` | `"a3b2"` |
| 2 | `"wwwwww"` | `"w6"` |
| 3 | `"abc"` | `"a1b1c1"` |
