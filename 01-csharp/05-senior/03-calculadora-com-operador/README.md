# 03 - Calculadora com operador

**Nivel:** Senior  
**Categoria:** Condicao

## Enunciado

Crie um programa que receba dois numeros e um operador (+, -, *, /, % ou ^) e devolva o resultado, tratando divisao por zero com ERRO.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `6, 2, "/"` | `3` |
| 2 | `2, 3, "^"` | `8` |
| 3 | `5, 0, "/"` | `ERRO` |
