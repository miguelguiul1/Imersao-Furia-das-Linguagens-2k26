# 08 - Rotacao de vetor

**Nivel:** Senior  
**Categoria:** Vetor

## Enunciado

Crie um programa que receba uma lista e um numero K, e mostre a lista rotacionada K posicoes para a direita.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `[1,2,3,4,5], K=2` | `[4,5,1,2,3]` |
| 2 | `[1,2,3], K=0` | `[1,2,3]` |
| 3 | `[1,2], K=3` | `[2,1]` |
