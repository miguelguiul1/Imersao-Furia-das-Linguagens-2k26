# 05 - Sequencia de Fibonacci

**Nivel:** Junior  
**Categoria:** Repeticao

## Enunciado

Crie um programa que receba N e mostre os N primeiros numeros da sequencia de Fibonacci.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `5` | `0 1 1 2 3` |
| 2 | `7` | `0 1 1 2 3 5 8` |
| 3 | `1` | `0` |

## Dica

Guarde os dois ultimos numeros e va somando para gerar o proximo.
