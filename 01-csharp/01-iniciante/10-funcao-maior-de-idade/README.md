# 10 - Funcao maior de idade

**Nivel:** Iniciante  
**Categoria:** Funcao

## Enunciado

Crie uma funcao que devolva verdadeiro se a idade for 18 ou mais, e mostre o resultado.

## Cenarios de teste

| Teste | Entrada | Saida esperada |
| --- | --- | --- |
| 1 | `20` | `verdadeiro` |
| 2 | `18` | `verdadeiro` |
| 3 | `15` | `falso` |

## Assinatura sugerida

`bool ehMaiorDeIdade(int idade)`

## Dica

Retorne diretamente o resultado da comparacao idade >= 18.
